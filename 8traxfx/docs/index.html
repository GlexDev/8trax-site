8TraxFX Documentation

8TraxFX is a Telegram-based marketplace built around:

  • Public Market Channel
  • Seller Pages
  • Discussion Room
  • Seller/Admin Bot

Buyers

  1. Browse listings at @EightTraxMarket
  2. Open seller page from listing
  3. Review details & requirements
  4. Ask questions in EightTrax_Main discussion room
  5. Proceed with seller directly

Buyers do NOT use the bot.

Sellers

  1. Use @EightTraxFXBot to submit listing
  2. Admin review
  3. Approved listing published to market channel
  4. Seller receives profile/store presence

Required Listing Info

  • Tool Type (EA / Indicator / Setfile)
  • Platform (MT4 / MT5)
  • Supported Pairs
  • Timeframes
  • Requirements (VPS / Broker / Deposit)
  • Clear Pricing
  • Support Terms

Marketplace Rules

  • No fake results
  • No stolen tools
  • No misleading claims
  • No spam
